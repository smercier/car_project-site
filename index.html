cartalib

librairie de cartes pour mapserver

Cartalib est une librairie de cartes réalisées par Mapgears pour ses projets, spécialement conçue pour Mapserver. Ces cartes sont directement utilisables via des API telles OpenLayers, leaflet ou D3.js ou encore Polymaps, par exemple. Cette librairie de carte ouverte offre une grande flexibilité et peux être librement utilisée à des fins non-commerciales. Elle peut être utilisé via un service Cartalib en ligne, en mode serveur autonome à l'aide du dépôt de fichiers directement copié sur votre serveur, ou enfin à l'aide d'un serveur dédié contenant l'installation complète de toutes les composantes technologiques.

Nous avons d'autre cartes prêtes à l'utilisation construite pour les données Navteq, Natural Earth et Tiger, nous contacter pour ce genre de besoin. Vous trouverez ici quelques indications pour vous permettre d'intégrer facilement cette carte à votre application.

Serveur dédié

Destinée à des utilisateurs qui ont fait le choix de Mapserver. comme moteur cartographique, cette librairie permet de profiter immédiatement d'une carte optimisée pour votre besoins. Le produit allie les données géographiques et des fichiers de configuration ("mapfiles") spécialement conçus et optimisés pour leur utilisation avec MapServer. Ce type d'installation nécésite l'installation d'une base données ouvertes OpenStreetMap à l'aide de l'utilitaire d'importation imposm. Vous pourrez trouver la précédure complète d'installation de toutes les composantes Open Source sur le wiki Rendering OSM data with MapServer. Ce produit est basé sur les travaux de Thomas Bonfort avec l'outil Basemap pour Mapserver. Vous pouvez télécharger le produit et l'utiliser. Nous pouvons aussi adapter le produit à votre besoin.
