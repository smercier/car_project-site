Cartalib OSM

Cartalib OSM est une librairie de cartes spécialement conçue pour Mapserver. Conçu pour des utilisateurs qui ont fait le choix de Mapserver. comme moteur cartographique, il permet de profiter immédiatement d'une carte OpenStreetMap optimisée pour votre besoins. Le produit allie les données géographiques provenant d'OpenStreetMap à des fichiers de configuration ("mapfiles") spécialement conçus et optimisés pour leur utilisation avec MapServer. Cette carte tuilée, produite avec des données ouvertes OpenStreetMap est un exemple de ce que nous offrons. Ce produit est basé sur les travaux de Thomas Bonfort avec l'outil Basemap pour Mapserver. Vous pouvez télécharger le produit et l'utiliser. Nous pouvons aussi adapter le produit à votre besoin.

Télécharger


Vous trouverez ici quelques indications pour vous permettre d'intégrer facilement cette carte à votre application.

leaflet

Avec l'api leaflet, vous pourrez utiser ce service de cette façon: 


   var tilejson = {
        tilejson: '1.0.0',
        tms: 'true',
        maxZoom: 18,
        attribution: 'Données © <a href="http://openstreetmap.org">OpenStreetMap</a> 
                      contributors, <a href="http://creativecommons.org/licenses/by-sa/2.0/">CC-BY-SA</a>,
                      Produit par <a href="http://mapgears.com">Mapgears</a>'
         };
   L.tileLayer('http://cartalib.mapgears.com/mapcache/tms/1.0.0/grayprint_v1@g/{z}/{x}/{y}.png', tilejson).

openlayers

Il est aussi possible d'utiliser l'api OpenLayers pour insérer cette carte à votre application sans aucune autre composante serveur particulière. Simplement télécharger (attention 2GB) et exposer l'arborescence de répertoires de cette carte sur votre serveur. Par la suite, utiliser un layers de type TileCachede cette façon.


var olLayer = new OpenLayers.Layer.TileCache("grayprint",
                ["http://cartalib.mapgears.com/tiles/",
                 "http://cartalib1.mapgears.com/tiles/",
                 "http://cartalib2.mapgears.com/tiles/",
                 "http://cartalib3.mapgears.com/tiles/",
		"http://cartalib4.mapgears.com/tiles/"],
                "grayprint",
                { name: 'grayprint',
