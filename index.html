cartalib

librairie de cartes pour mapserver

Cartalib est une librairie de cartes pour mapserver. Ce produit est destiné à ceux qui désire installer Mapserver dans le réseau interne de leur organisation et avoir accès à des cartes préconfigurées d'avance. Tout ce qu'il ne reste à faire est de copier les fichiers de configuration de la carte sur votre serveur. Ce produit est basé sur les travaux de Thomas Bonfort avec l'outil Basemap pour Mapserver

View on GitHub Download .zip

Nous avons produit cette carte avec des données ouvertes Open Street Map . Vous trouverez ici quelque indications pour vous permettre de l'intégrées facilement à votre application. Cette carte n'est siponible que pour le Québec actuellement.

leaflet

Avec l'api leaflet, vous pourrez utiser ce service de cette façon: 


   var tilejson = {
        tilejson: '1.0.0',
        tms: 'true',
        maxZoom: 18,
        attribution: 'Données © \OpenStreetMap\ contributors, \CC-BY-SA\, Produit par \Mapgears\'
         };
   L.tileLayer('http://cartalib.mapgears.com/mapcache/tms/1.0.0/grayprint_v1@g/{z}/{x}/{y}.png', tilejson).
